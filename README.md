# Siemens-CleanCode-A1


## Instructions


### Download


```shell
https://github.com/RoniEmad/Siemens-CleanCode-A1.git
cd Siemens-CleanCode-A1/
```

### Build

To build this project, you can use the following command:

```shell
g++ -g Assignment-Answer.cpp -o cleancode
```
### Run
```shell
./cleancode
```
### Debug
```shell
gdb cleancode
```
